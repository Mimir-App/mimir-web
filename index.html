Capture :9476

mimir-capture

Servicio systemd que corre siempre en background. Captura la actividad del escritorio via xdotool y D-Bus, y escribe bloques en SQLite.

  • Poller cada 30 segundos
  • Deteccion de proyecto git
  • Tray icon con control de modo
  • Recuperacion tras crash
Server :9477

mimir-server

API FastAPI lanzada por la app al abrirse. Gestiona integraciones con sistemas de gestion, repositorios y proveedores de IA.

  • CRUD de bloques
  • Integraciones extensibles
  • Scoring y priorizacion
  • Descripciones con IA

Instalacion rapida

# 1. Instalar la app (incluye mimir-server)
sudo apt install ./mimir_0.3.0_amd64.deb

# 2. Opcional: instalar captura automatica
sudo apt install ./mimir-capture_0.3.0_amd64.deb
systemctl --user enable --now mimir-capture

# 3. Verificar
curl http://127.0.0.1:9476/health

Mimir Server

Servidor central para equipos y empresas

Mimir Server permitira a las organizaciones gestionar las integraciones, licencias y configuraciones de forma centralizada. Cada empleado usara la app de escritorio con la configuracion que su empresa haya definido, sin que ningun dato personal sea visible para la administracion.

  • Panel de administracion centralizado
  • Gestion de integraciones por empresa (Odoo, GitLab, GitHub...)
  • Configuracion de etiquetas de prioridad y scoring
  • Dashboards agregados por equipo y departamento
  • Control de licencias y servicios habilitados
  • Despliegue on-premise o en la nube
